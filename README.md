# Power_BI_Sales_Dashboard_Onitiju
Power BI Sales Dashboard. Interactive data visualization and business insights dashboard built with Power BI using sales order data.

# Project Title

Power BI Sales Performance Dashboard

# Project Overview

This project presents an interactive Sales Performance Dashboard developed using Power BI to analyze sales transactions, customer behavior, and order fulfillment performance. The dashboard provides key insights into sales trends, order status, shipping performance, and customer distribution.

The objective of this project is to demonstrate practical data analysis, data modeling, and visualization skills for business decision-making.
