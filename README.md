# Power_BI_Sales_Dashboard_Onitiju
Power BI Sales Dashboard. Interactive data visualization and business insights dashboard built with Power BI using sales order data.

# Project Title

Power BI Sales Performance Dashboard

# Project Overview

This project presents an interactive Sales Performance Dashboard developed using Power BI to analyze sales transactions, customer behavior, and order fulfillment performance. The dashboard provides key insights into sales trends, order status, shipping performance, and customer distribution.

The objective of this project is to demonstrate practical data analysis, data modeling, and visualization skills for business decision-making.

# Key Features

Interactive sales performance dashboard
Order status tracking (Delivered, Shipped, Processing, Cancelled)
Monthly sales trend analysis
Customer order distribution
Shipping performance analysis
Dynamic date slicers and filters
Clean and intuitive data visualization

# Tools & Technologies Used

Power BI Desktop
Microsoft Excel / CSV Dataset
Data Modeling
DAX (Data Analysis Expressions)
Data Cleaning and Transformation (Power Query)

# Dataset Information

The dataset contains simulated sales order data with the following fields:

Order ID
Customer ID
Order Date
Shipping Date
Order Status
Shipper ID
Comments
The dataset was used to demonstrate data visualization and business intelligence techniques.

# Dashboard Insights

The dashboard helps answer key business questions such as:
What is the monthly sales trend?
What percentage of orders are successfully delivered?
Which customers generate the most orders?
How efficient is the shipping process?
